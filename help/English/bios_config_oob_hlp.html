BIOS Configurations - Help

BIOS Configuration provides a method to configure "BIOS Setup Variables" through BMC EWS (This sub method will help to configure only this particular BIOS Configurations, there are other options available to configure other "BIOS Setup Variables" ).

Select a BIOS Variable

Select a BIOS Variable from the drop down menu.

BIOS Variable Value

Once a BIOS Variable is selected using above option (Select a BIOS Variable), the corresponding BIOS Variables Current Value will be displayed in drop down box. Other available options for the corresponding BIOS Variable can be seen by clicking on the drop down box and if value needs to be changed for the above variable then, other available values can be selected from the dropdown box .

Key Value

Selected BIOS Variable Value.

BIOS Variable Description

BIOS Variable Value correspond to configure parameter value.

Value

Current selected BIOS Variable Value.

SavedValue

Saved parameter value in the system.

Save

Click the "Save" button, once the BIOS Variable Value has been chosen from the dropdown box and changed value will get reflected in Grid Table .

Cancel

Click "Cancel" button, if you want to restore the earlier value ,only if the bios variable is not saved ( i.e, before clicking "Save" button.). If the bios variable is saved by clicking "Save" button, then clicking "Cancel" button will not restore the earlier value.

NOTE:

Changed BIOS Variable Values will get reflected in BIOS, only during next BIOS reboot .
