CPLD Update - Help

Use this page to upload new CPLD firmware.

CPLD Update

Current CPLD firmware version information is available for viewing, as well as the option to select and upload or drag and drop a new CPLD firmware image. By dropping a new CPLD firmware image on the page or selecting upload, the BMC will begin its CPLD firmware image upload process, which will take a couple of minutes. The alert message will be displayed after the firmware image is uploaded successfully.

If CPLD Update Option - Reset Immediately is selected, the BMC will reboot automatically. As all web sessions will be terminated on a BMC reboot, you will be required to log in again to verify that the CPLD firmware update was successful.

If CPLD Update Option - Reset Immediately is not selected, You will be required to reboot the BMC manually(DC cycle) to verify that the CPLD firmware update was successful.

CPLD FW Rev

Displays the current CPLD firmware version.

CPLD Update Options

Recovery - Select the recovery option, if you want to update CPLD firmware with the recovery option.

Reset Immediately - Select Reset Immediately to reboot the BMC after the CPLD firmware image is uploaded successfully.
