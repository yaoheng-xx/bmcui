BIOS/ME Firmware Update - Help

Use this page to upload new firmware(BIOS/ME/FD).

BIOS/ME Firmware Update

The image version information is available for viewing, as well as the option to select and upload or drag and drop a new firmware image. By dropping a new image on the page or selecting upload, web service begins its firmware update process, which will take a couple of minutes. When it is finished, it will store the image inside BMC. Perform the update server reboot (DC cycle), BIOS will mount the image as a USB virtual media image.

BIOS Rev

Displays the current BIOS version.

Mgmt Engine (ME) FW Rev

Displays the current ME firmware version.

PCH PFM Active Rev

Displays the current PCH PFM active version.

PCH PFM Recovery Rev

Displays the current PCH PFM recovery version.

Backup

When the option is enabled, the Backup region of current BIOS will be updated together.

NVRam

When the option is enabled, the NVRam region of current BIOS will be updated together..

DeferReset

When the option is enabled, the Host System reset will be deferred.

Password

The option will appear when "Enforce Password Mode" is Enabled in BIOS. Users should input password before uploading the image.
