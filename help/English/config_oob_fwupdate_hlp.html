BIOS/IFWI Firmware Update - Help

Use this page to upload new firmware(BIOS/IFWI).

BIOS/IFWI Firmware Update

The BIOS/IFWI Firmware image version information are available for viewing, as well as the option to select and upload or drag and drop a new BIOS/IFWI firmware image. By dropping a new BIOS/IFWI firmware image on the page or selecting upload, web service begins its BIOS/IFWI firmware image upload process, which will take a couple of minutes.

If BIOS Update Option - Reset Immediately is selected, BMC will reboot automatically. As all web sessions will be terminated on a BMC reboot, you will be required to log in again to verify that the BIOS/IFWI firmware update was successful.

If BIOS Update Option - Reset Immediately is not selected, it will store the image inside the BMC. Perform the update server reboot (DC cycle), BIOS will mount the image as USB virtual media and the image.
