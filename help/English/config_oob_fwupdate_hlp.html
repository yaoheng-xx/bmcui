BIOS/IFWI Firmware Update - Help

Use this page to upload new firmware(BIOS/IFWI).

BIOS/IFWI Firmware Update

The BIOS/IFWI Firmware image version information is available for viewing, as well as the option to select and upload or drag and drop a new BIOS/IFWI firmware image. By dropping a new BIOS/IFWI firmware image on the page or selecting upload, the web service begins its BIOS/IFWI firmware image upload process, which will take a couple of minutes.

If BIOS Update Option - Reset Immediately is selected, BMC will reboot automatically. As all web sessions will be terminated on a BMC reboot, you will be required to log in again to verify that the BIOS/IFWI firmware update was successful.

If BIOS Update Option - Reset Immediately is not selected, it will store the image inside the BMC. Perform the update server reboot (DC cycle), BIOS will mount the image as a USB virtual media image.

BIOS Rev

Displays the current BIOS version.

Mgmt Engine (ME) FW Rev

Displays the current ME firmware version.

BIOS Update Options

Recovery - Select the recovery option, if you want to update BIOS/IFWI Firmware with the recovery option.

Reset Immediately - Select the Reset Immediately to reboot the BMC after the BIOS/IFWI firmware image is uploaded successfully.
