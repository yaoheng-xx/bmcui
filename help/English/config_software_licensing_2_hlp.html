Advanced System Management Key - Help

Use this page to upload new advanced system management key.

You can view the last upload time of the advanced system management key and the options for selecting and uploading or dragging and dropping new advanced system management key files. By placing a new advanced system management key on the page or choosing to upload, the advanced system management key will begin the update process, which will take some time. Progress will continue to be reported until the end of the update.

Last Upload Time

Show the last upload time of the advanced system management key.

Activated Features

You can view the activation status of the advanced functions before and after the advanced system management key is uploaded. Advanced features include BMC Virtual Media, Active Directory.

Actions
Drop a file on this page / Choose File / Browse

Drop a file on this page or choose Choose File / Browse to upload.

Upload

Upload the advanced system management key to the BMC for update action.
