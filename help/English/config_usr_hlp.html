Users - Help

The list of configured users, along with their status and network privilege is displayed.

Use this page to configure the IPMI users and privileges for this server.

User ID

The ID of the user.

User Name

The account of the user.

User Status

The enable/disable status of the user.

Network Privilege

The network privilege status of the user.

NOTE: If UserID 1 (root) is available, It may not be renamed or deleted.

Actions
Add User

Select an empty slot in the list and click to add a new user.

Modify User

Select a user in the list and click to modify their settings.

Delete User

Select a user in the list and click to delete.
