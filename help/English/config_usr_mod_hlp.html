Modify User - Help

Use this page to modify a user's password and permissions.

User Name

The name of the user being configured.

The name is a string of 1 to 16 alpha-numeric characters or '.','_' or '-'.

The name must begin with an alpha-numeric character or '_'.

Change Password

To change the user's password, check this box which will enable the password fields.

Password, Confirm Password

Space at the beginning is not allowed.

System has Disabled, Low, Medium and High Password Complexity mode.

Users can change system password mode and check the password complexity rules on the "Security Settings" page.

Enter and confirm the new password here.

Network Privilege

The level of network privilege to assign to this user.

  • - Administrator
  • - Operator
  • - User
  • - No Access

Enable Access

Check this box to enable the user for LAN access. Also enables/disables SOL, KVM, Media Redirection.

Click here to get detailed network privilege of each level
Actions
Modify

Click 'Modify' to accept the modifications and return to the user list.

Cancel

Click 'Cancel' to return to the list without modifying the new user.
