IPv6 Network Settings - Help

Use this page to configure the IPv6 network settings for server management.

Interface

Lists the Interface available for server management.

MAC Address

The MAC address of the device (read only).

IPMI Channel

Display IPMI Channel number (read only).

NIC Description

Display NIC dedicated to BMC / Host or shared between Host and BMC of Interface (read only).

Link Status

Display NIC Link status of Interface (read only).

Obtain an IP address automatically (use DHCPv6/SLAAC)

Select the option to enable the IPv6 DHCP. IP Address, Prefix Length, Gateway and DNS Server input fields will be disabled if "Obtain an IP address automatically (use DHCPv6/SLAAC)" option is selected.

Use the following IP address

Select the option to configure the static IPv6 address. IP Address, Prefix Length, Gateway and DNS Server input fields will be enabled to configure if "Use the following IP address" option selected

IP Address

Select the type of IP assignment with the radio buttons. If configuring a static IP, enter the requested IP Address, IPv6 prefix length, and optionally the Gateway, Primary DNS and Secondary DNS Server in the given fields.

  • - IPv6 addresses consist of 8 4 digit hexadecimal numbers separated by colons.
  • - A :: can be used for a single sequence of two or more zero fields.

Prefix Length

Display the details of Prefix Length.

Gateway

Denotes the Gateway details of the device.

Primary DNS Server

Displays the Primary DNS Server details.

Secondary DNS Server

Displays the Secondary DNS Server details.

Actions
Save

Click 'Save' to save any changes made.

NOTE: The Dedicated Management NIC IP address must be on a different subnet than the baseboard IP address used for management traffic.


NOTE: Please close any active KVM sessions prior to changing the network settings, in order to avoid disconnection from the server.
