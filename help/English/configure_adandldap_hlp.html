To enable or configure the Active Directory server, please select the checkbox.

Enable Active Directory Authentication

Select the checkbox to enable/disable Active Directory Authentication.

Active Directory Authentication over SSL

Select the checkbox to enable/disable Active Directory Authentication over SSL.

Certificate is common for both LDAP & AD.

CA Certificate valid until -- It denotes the expiry date for CA Certificate.

IP Address

The IP address of AD server.

  • - IP Address made of 4 numbers separated by dots as in "xxx.xxx.xxx.xxx".
  • - 'xxx' ranges from 0 to 255.
  • - First 'xxx' must not be 0.
  • - Supports Host Name

Bind DN

The Distinguished Name of the AD server, e.g. "cn=Manager, dc=my-domain, dc=com".

Bind Password

Authentication password for AD server; the password must be at least 4 characters long.

Searchbase

The searchbase of the AD server, e.g. "dc=my-domain, dc=com".

Timeout

The Timeout of the AD Server

User ID Attribute (Optional)

User ID for the AD Server

Port (Optional)

The Port Number of the AD Server
