Alerts - Help

When one of the selected system events occurs, an alert is generated and sent to the configured destination(s).

Select the events that will trigger alerts

Select one or more system events that will trigger an Alert. Clearing all events disables Alerts. These events correspond to the IPMI preconfigured Platform Event Filters.

Alert Destinations

Alert destinations can be one of two types:

  • Email (requires Alert Email to be configured)

Actions
Check All

Check all events to generate Alerts.

Clear All

Uncheck all events so no Alerts will be generated.

Save

Click 'Save' to save any changes made.

Send Test Alert

To test whether an alert will reach its destination, configure at least one destination. Then click 'Send Test Alerts' to send a simple test alert to the destination(s).
