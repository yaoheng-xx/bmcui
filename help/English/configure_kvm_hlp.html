KVM & Media - Help

Use this page to allow changes to the KVM port during a redirect session and configure virtual media timeout in seconds.

Default Ports:

This section allows setting the ports used by KVM and remote media (only secure ports). Do not change these values unless you know the new ports are unused.

Note:

If the singleport feature is enabled we should not update the port number. It will use the web secure port as default.

Virtual Media Timeout:

This section allows configuring the virtual media timeout in seconds.

Note:

If the virtual Media redirection is in progress, we should not update the timeout settings.
