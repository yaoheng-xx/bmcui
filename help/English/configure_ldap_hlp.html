LDAP - Help

To enable/disable LDAP, check or uncheck the "LDAP Authentication" checkbox respectively.

LDAP Authentication

Check this box to enable LDAP authentication, then enter the required information to access the LDAP server.

LDAP Authentication over SSL

Select the checkbox to enable/disable LDAP Authentication over SSL.

IP Address

The IP address of the LDAP server.

  • - IP Address made of 4 numbers separated by dots as in "xxx.xxx.xxx.xxx".
  • - 'xxx' ranges from 0 to 255.
  • - First 'xxx' must not be 0.
  • - Supports Host Name

Bind DN

The Distinguished Name of the LDAP server, e.g. "cn=Manager, dc=my-domain, dc=com".

Bind Password

Authentication password for LDAP server; the password must be at least 4 characters long.

Searchbase

The searchbase of the LDAP server, e.g. "dc=my-domain, dc=com".

User ID Attribute (Optional)

User ID for the LDAP Server

LDAP Group Configuration - Help

Group Name

Role Group Name is a string of 64 alpha-numeric characters.

Special symbols hyphens and underscores are allowed.

Group Privilege

Select the Role Group Privilege. This is the level of privilege to be assigned to this role group

Actions
Save

Save the current settings.

Delete

Select the LDAP Group from the table and Delete the Group.
