VLAN - Help

Use this page to configure an 802.1Q VLAN private network on the specified Physical Interface.

Physical Interface

Lists the Physical Interface(s) available for server management. The Physical Interfaces describe the physical NIC connection on the server. All channels are onboard NICs. The Baseboard Mgmt channel is a shared NIC configured for management and shared with the operating system.

VLAN List

Displays all the VLAN info in table format for the selected LAN Channel.

Virtual Interface - Configured Virtual Interface name will be displayed.

VLAN ID - Configured VLAN ID will be displayed.

  • - Create new VLAN information. Please configure the following fields' VLAN ID and click the "Add" button to add a VLAN virtual interface.
  • - To delete the existing VLAN. Please click the delete icon at the end of the existing VLAN row.

VLAN ID

Specify the VLAN ID to use. Values are from 1 to 4094.

Actions
Add

Click 'Add' to add a virtual interface to the selected LAN channel.
