BMC Firmware Update - Help

Use this page to update BMC firmware.

BMC Firmware Update

Current BMC firmware version information is available for viewing, as well as the option to select and upload or drag and drop a new firmware image.

By dropping a new firmware image on the page or selecting upload, the BMC will begin its firmware file upload process, which will take a couple of minutes. Firmware file upload progress will be displayed.

NOTE:

By default, BMC Update Option - Reset Immediately will be unchecked. So after the firmware image uploaded successfully user can do other operations and need to restart the BMC manually(DC cycle) to check the new firmware.

If you want to reset the BMC Immediately after firmware image uploaded, Select the BMC Update Option - Reset Immediately before uploading new BMC firmware image.

BMC FW Rev

BMC Update Options
