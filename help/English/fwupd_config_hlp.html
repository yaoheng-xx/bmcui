Firmware Update

Current firmware version information is available for viewing, as well as the option to select and upload or drag and drop a new firmware image. By dropping a new firmware image on the page or selecting upload, the BMC will begin its firmware update process, which will take a couple of minutes, finishing with the BMC rebooting to run the new firmware. Progress will be reported up until the time of reboot, after which it will take about one minute for the embedded web server to start responding again. As all web sessions will be terminated on a BMC reboot, you will be required to log in again to verify that the firmware update was successful.
