NM Configuration - Help

Configure node manager policy settings.

Policy ID

The policy ID to add/edit/delete. The valid range is 0-255.

In the policy table, policy numbers with an asterisk (*) are policies set externally using a non-platform domain. Changing parameters on these policies will not affect their triggers, trigger limits, reporting periods, correction timeouts, or aggressive CPU throttling settings.

Timers

If enabled, you may configure policy suspension periods. Each policy may have up to five suspension periods.

Suspend periods are repeatable by day-of-week. Start and stop times are designated in 24-hour format, in increments of 6 minutes. To specify a suspension period crossing midnight, two suspension periods must be used.

Enable

Check this box if the policy is to be enabled immediately.

Shutdown

Check this box to enable a system shutdown if the policy is exceeded and cannot be corrected within the correction timeout period. The operating system will be given 30 seconds to shut down gracefully. If the system is still not shut down after 30 seconds, the BMC will initiate an immediate shutdown.

Alert

It displays whether the log event is enabled/disabled (0 - Disabled, 1 - Enabled)

Power Limit (Watt)

The desired platform power limit is in watts.

Add Node Manager Policies

Input the policy number.

Setting Enable Policy: Disable/Enable.

Setting Policy Exception Action: No Action, Alert (Log Event) and/or Shutdown.

Setting Power Limit(Watt). Valid range will be displayed in the alert if invalid power limit is given.

Click the 'Save' button to save.

Log Event

Check this box to enable the node manager to send a platform event message to the BMC when a policy is exceeded.

Edit Node Manager Policies

Select an existing policy from Policy Table.

Setting Enable Policy: Disable/Enable.

Setting Policy Exception Action: No Action, Alert (Log Event) and/or Shutdown.

Setting Power Limit(Watt). A valid range will be displayed in the alert if an invalid power limit is given.

Click the 'Save' button to save.

Delete Node Manager Policies

Select an existing policy from Policy Table.

Click the 'Delete' button to delete.

Use Policy Suspend Timers

Select the 'Yes' radio button if you want to enable Policy Suspend Timers.

Choosing the time periods that you want to suspend the NM policy.

Click the 'Save' button to save.

Actions
Save

Click 'Save' to save any changes made.

Delete

Select a policy in the list and click to delete.

Cancel

Click 'Cancel' to discard changes.
