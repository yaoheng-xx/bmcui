Event Log - Help

The Event Log displays all System Event Log records captured by the BMC.

Each SEL record is tagged with an Event ID, Timestamp, Sensor Name, Controller, Severity, Sensor Type and Description field.

The Event Log displays all records present by default. The number of records displayed in one page can be controlled. It is recommended the number of records per page be kept to a minimum to keep page load times small. The event records may be filtered by the source of the event, as well as the severity of the event.

Severity category

  • Informational
  • Warning
  • Critical

Total Event Log

The System Event Log is able to store 4000 event records. A warning message is displayed once the event log has filled. New records added to the Event Log cause the oldest records to be discarded.

Timestamp

The timestamp of the system event log.

Severity

The severity of the system event log.

Description

The description of the system event log.

Number of entries per page

Select the dropdown options to display the number of event logs

Actions
Clear Event Log

Click on the Clear Event Log button to remove all Event Log entries.

Save Event Log

Click on the Save Event Log button to save all of the records in the Event Log. The records are compressed into a ZIP file. A Save File dialog allows you to choose where the compressed file is written.

Refresh Event Log

Click on the Refresh Event Log button to redisplay all of the current set of event records.The Event Log does not automatically refresh, in order to allow review of the entries.
