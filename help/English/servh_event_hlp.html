Event Log - Help

The Event Log page displays the server management event log

Severity Category

  • Informational: Operating normally.
  • Warning: A condition that requires attention.
  • Critical: Requires immediate attention.

Total Event Log

The System Event Log is able to store thousands of event records. The earliest hundreds of events will be discarded when the event log space is full.

Timestamp

The date and time when the log entry was created.

Severity

The severity of the system event log.

Description

The message of the log entry.

Number of entries per page

Specify how many events are displayed per page.

Actions
Clear Event Log

Clear the event log.

Save Event Log

Save the event log entries to a JSON file EVENT_LOG.json.

Refresh Event Log

Refresh the event log, this page does not automatically refresh
