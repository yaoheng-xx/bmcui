Sensor Readings - Help

Use this page to see the sensor readings and their thresholds.

Select a sensor type category

Select the type of sensor readings to display in the list. The default is to see all sensors.

Auto Refresh(sec)

It provides eight options for selection.

Sensor Readings list

Selected sensors will be shown with their healthy, name, status, and readings.

Click 'Show Thresholds' to expand the list, showing low and high threshold assignments. Use the scroll bar at the bottom to move the display left and right.

Click 'Hide Thresholds' to return to the original display, hiding the threshold values.

Healthy

The sensor health status on the platform.

Name

The sensor name on the platform.

Status

The sensor status on the platform.

Reading

The sensor readings on the platform.

Low NR

The lower non-recoverable values of the sensor.

Low CT

The lower critical values of the sensor.

Low NC

The lower non-critical values of the sensor.

High NC

The upper non-critical values of the sensor.

High CT

The upper critical values of the sensor.

High NR

The upper non-recoverable values of the sensor.

Sensor Live Reading

Click the sensor from the list to show the sensor live reading chart and Logs

Actions
Refresh

Click to refresh the selected sensor readings.

Show Thresholds

Click to see non-critical, critical and non-recoverable thresholds for the selected sensors.

Hide Thresholds

Click to see only the name, status and reading for selected sensors.
