Adapter - Help

Use this page to display the controller information of RAID/HBA.

Firmware Version

Displays the firmware version of the selected Storage Controller (RAID card).

Member ID

Displays the Member ID of the selected Storage Controller.

Name

RAID Controller (RAID card) name.

Serial Number

The serial number for the selected StorageController (RAID card).

Supported RAID Types

Describes the RAID Types supported by the StorageController (RAID card).

Note: Supported RAID Types are not available for HBA Controllers.

Status

Displays the Health status of the selected RAID.

Actions
View Physical Devices

Click 'View Physical Devices' redirected to the Physical Devices page.

View Logical Devices

Click 'View Logical Devices' redirected to the Logical Devices page and this button functionality is not applicable to HBA Controller.
