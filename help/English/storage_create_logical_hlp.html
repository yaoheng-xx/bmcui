Create Logical Device - Help

Use this page to create the logical Device of the RAID controller.

RAID Controller

Select the Dropdown to choose the RAID Controller

RAID Level

Select the dropdown to choose the Primary RAID Level

Span Depth

Enter the depth of the Span in the Span Depth field.

Number of Drives

Enter the number of Drives in the Drives per Span field.

Virtual Device Name

Enter the Virtual Name of the Device.

Initialization State

Select the Initialization type from the drop-down lists.

Stripe Size

Enter the Stripe Size of the virtual device

Read Policy

Select the dropdown to choose the read policy

Write Policy

Select the dropdown to choose the written policy

IO Policy

Select the dropdown to choose the IO policy

Access Policy

Select the dropdown to choose the access policy

Disk Cache Capacity

Select the dropdown to choose the Disk Cache Capacity

UnConfigured Physical Drives

Select UnConfig Physical Drives from the drop-down list. To select multiple physical devices, Hold down the Ctrl key and select from the list.

Note: Physical Drives which are already configured to the Logical Device won't display in the UnConfigured Physical Drives dropdown. UnConfigured Physical Drives should be equal to multiples of Span Depth and Drives per Span.

Actions
Create

Click Create action button to create the Logical Volume of the Device.

Cancel

Click Cancel to return to the Logical Device information page.
