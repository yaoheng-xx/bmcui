Logical Device Information - Help

Use this page to display the logical Device Information of RAID controller.

Name

It displays the name of Logical Device.

RAID Type

It displays details of RAID type

Strip Size Bytes

Its denotes the details of Strip Size Bytes.

Read Cache Policy Type

It displays the details of read cache policy type.

Write Cache Policy Type

It shows the details of write cache policy type.

Capacity Bytes

It displays the details of capacity bytes.

Number of Physical Devices

Displays the details of Number of physical devices.

Actions
Create Logical Device

To Create the new Logical devices Click the 'Create Logical Device' action button. It will redirect to Create Logical Devices page.

Delete

Click the Delete icon and confirm the alert message to delete the selected Logical Device.
