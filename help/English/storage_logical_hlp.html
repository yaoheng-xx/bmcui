Logical Device Information - Help

Use this page to display the logical Device Information of the RAID controller.

Name

Displays the name of the Logical Device.

RAID Type

The RAID type of the Logical Device.
Allowed value: "RAID0", "RAID00", "RAID1", "RAID10", "RAID1E", "RAID5", "RAID50", "RAID6", "RAID60"

Strip Size Bytes

The number of blocks (bytes) in a strip in a disk array that uses striped data mapping.
Allowed value: "64K", "128K, "256K", "512K", "1MB" etc.

Read Cache Policy Type

Indicates the read cache policy setting for the Logical Device.
Allowed value: "NoReadAhead", "ReadAhead"

Write Cache Policy Type

Indicates the write cache policy setting for the Logical Device. Allowed value: "WriteThrough", "WriteBack", "AlwaysWriteBack"

Capacity Bytes

The size of the smallest addressable unit (Block) of this Logical Device in bytes.

Physical Devices IDs

Displays the Number of physical devices mapped to this Logical Device.

Actions
Create Logical Device

To Create the new Logical devices, Click the 'Create Logical Device' action button. It will redirect to Create Logical Devices page.

Delete

Click the Delete icon and confirm the alert message to delete the selected Logical Device.
