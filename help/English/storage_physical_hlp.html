Physical Device Information - Help

Use this page to display the Physical Device Information of the RAID/HBA controller.

ID

ID Column will display the Physical Device ID of the Device.

Failure Predicted

Displays whether the Physical Device is currently predicting a failure in the near future or not.

Block Size Bytes

The size of the smallest addressable unit (Block) of this Physical Device in bytes.

Manufacturer

To display the Manufacturer details of the Physical Device.

Negotiated Speed Gbs

The speed of the Physical Device is currently communicating to the StorageControllers in Gigabits per second.

Protocol

The protocol of the Physical Device is used to communicate with the StorageControllers.

Revision

The revision of this Device. This is typically the firmware/hardware version of the Physical Device.

SerialNumber

Displays the Serial number of the Physical device.

Status

Its shows the Status of the device.

Slot Number

The Slot Number Column will provide the information about the Drive Slot of the Physical Device.
