CPU Information - Help

Use this page to view information about the processor(s) installed in the server.

The data in the CPU Asset Information page is collected from SMBIOS entries sent from the BIOS to the Baseboard Management Controller at the end of POST. If there is no data available, or the data is stale, please reset the system, allow it to complete POST, and refresh the page using the refresh button above.
