System Information - Help

Use this page to view information about the server.

Host Power Status

Shows the power status of the host (on/off).

Advanced Management Key

Shows whether the advanced management functions are activated.

Device (BMC) Available

Indicates whether the BMC is available for normal management tasks.

BIOS ID

BIOS ID:BoardFamilyID.OEMID.MajorVer.MinorVer.RelNum.BuildDateTime.

BMC FW Rev

Major and minor revision of the BMC firmware.

BMC Firmware Build Time

The date and time of the installed BMC firmware.

Backup BMC FW Rev

Major and minor revision of the Backup BMC firmware.

CPLD FW Rev

Major and minor revision of the CPLD firmware.

Mgmt Engine (ME) FW Rev

Major and minor firmware revision for the Management Engine (ME). Only available if the host is powered on.

Baseboard Serial Number

The Serial number of this specific baseboard.

Overall System Health

Overall system health LED status is displayed.

  • LED Solid Green - System Health LED
  • LED Blinking Green - System Degraded LED
  • LED Blinking Amber - Non-critical fault LED
  • LED Amber - Critical fault LED
  • LED Blue - Identify LED

Web Session Timeout

Set the maximum web service timeout in seconds. Timeout should be between 30 mins and 1 day. The Default timeout is 30 mins.

  • 1. Pull down the time-out values bar 30(mins)/1(hour)/2(hours)/4(hours)/8(hours)/1(day) to configure the value.
  • 2. After selection, this web session time-out value will take effect.
  • 3. Once the web session is time-out, the system will automatically redirect the user to the web login page.
