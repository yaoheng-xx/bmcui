NVMe Information - Help

This page displays status information for each Intel ® Corporation NVMe drive installed into a Hot Swap drive slot. Each group box displayed contains statistics for a single NVMe drive. The group box identifies the drive by Hot Swap Backplane (HSBP) number, and by drive slot number. Only five of the entries are able to change state at runtime:

Model: Indicates the current model serial number.

PCIe 0 Link Speed: It displays the link speed of PCIe 0.

PCIe 1 Link Speed: It displays the link speed of PCIe 1.

NVMe Powered: Indicates the current power on state of the NVMe drive. This value can only be presented when the NVMe drive is powered.

NVMe Reset Required: Indicates if the NVMe drive is reporting a condition requiring the drive to be reset.

Device Class: It denotes the details of Device Class.

Device Programming Intfc: It displays the details of Device Programming Intfc.

Port 1 PCIe Link Active: It displays the details of Port 1 PCIe Link Active.

Device Location: It shows the details of Device Location.

Serial Number: It displays the details of Serial Number.

PCIe 0 Link Width: It shows the details of PCIe 0 Link Width.

PCIe 1 Link Width: It indicates the details of PCIe 1 Link Width.

NVMe Functional: It displays the details of NVMe Functional.

Port 0 PCIe Link Active: It indicate the details of Port 0 PCIe Link Active status.

Device Sub-class: It displays the details of Device Sub-class.

Drive Life Consumed: It indicates the details of Drive Life Consumed.

Manufacturer: It display the details of Manufacturer.

Firmware revision: It denotes the details of Firmware revision.
