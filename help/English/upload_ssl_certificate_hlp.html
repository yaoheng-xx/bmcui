SSL Certification - Help

The details of the available Certificates are shown in the table. To upload a new SSL certificate, use the Add New Certificate button and Select the certificate type and navigate to the certificate and press the Upload button.

To Generate CSR code, Click Generate CSR button and fill the valid details and press the Generate Button.

Certificate

It shows the details of certificates.

Issued by

Denotes the details of Issued certificate issued by whom.

Issued to

It denotes the details of certificates issued to whom.

Valid from

It shows the valid starting date details.

Valid until

It shows the valid expiry date details.

Certificate Type

Select the certificate type to upload, ( Https, LADP and CA are supported )

Click "Choose File" button to certificate file

Actions
Add New Certificate

Click "Add New Certificate" button to show the upload options

Upload

Click "Choose File" button to setting new SSL certificate

Click "Upload" button

Generate CSR

Click "Generate CSR" button to show the CSR input fields

Click "Generate" button to generate the CSR code.

Replace

Click the replace icon from the SSL certificate table and choose the file and click replace

Delete

Click the delete icon from the SSL certificate table to delete the individual certificate

NOTE:
HTTPS and LDAP certificate can only be replaced.
Deleting HTTPS and LDAP Certificate is not supported.
