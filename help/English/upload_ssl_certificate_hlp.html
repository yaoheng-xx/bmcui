SSL Certification - Help

The certificates include server certificate and CA certificate.

Server Certificate

It shows the details of the server certificate. Server certificate can only be replaced. Deleting server certificate is not supported.

Certification Valid From

It shows the valid starting date details.

Certification Valid Until

It shows the valid expiry date details.

CA Certificate

It shows the details of the CA certificates. No CA certificate exist by default, please upload CA certificate if needed.

Issued by

Denotes the details of the Issued certificate issued by whom.

Issued to

It denotes the details of certificates issued to whom.

Valid from

It shows the valid starting date details.

Valid until

It shows the valid expiry date details.
