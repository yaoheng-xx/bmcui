Local Image - Help

This option allows you to share an ISO or IMG/IMA image over html5.

Each image will be emulated to the host as a USB device.

Actions
Launch Window to Mount Local Image

Click to launch the operation window for mounting the local image.

Device 1

Click to mount a drive image to the first mount point.

Device 2

Click to mount a drive image to the second mount point.

Select type

Click to select the drive image type: ISO or IMG/IMA.

Select media

Choose a drive image file.

Plug in

Plug in a drive image file.
