WEB ISO

This page allows you to share a ISO image using the SMB protocol. It allows you to share the image over a Windows Share or Linux Samba. This image will be emulated to the host as a USB device.

Note:

Only currently configured image can be able to mount and unmount.

To mount and unmount different image, Change the configuration and save.

Image name with following characters are not supported.

[{}()<>&*`|=?;[\]$\-#~!"%/\\:+,']
