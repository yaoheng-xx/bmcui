BIOS/IFWI 固件更新

BIOS/IFWI 固件映像版本信息可供查看，以及选择和上传或拖放新的 BIOS/IFWI 固件映像的选项。通过在页面上放置新的 BIOS/IFWI 固件映像或选择上传，Web 服务开始其 BIOS/IFWI 固件映像上传过程，这将需要几分钟。

如果选择了 BIOS 更新选项 - 立即重置，BMC 将自动重启。由于所有 Web 会话将在 BMC 重新启动时终止，因此您需要再次登录以验证 BIOS/IFWI 固件更新是否成功。

如果没有选择 BIOS 更新选项 - 立即重置，它会将映像存储在 BMC 中。执行更新服务器重启（DC 循环），BIOS 会将映像挂载为 USB 虚拟媒体和映像。
