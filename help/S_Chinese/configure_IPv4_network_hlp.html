IPv4 网络设置 — 帮助

使用此页面为服务器管理配置网络设置。

主机名称

主机名称是一个符合RFC 1123，少于64个字母数字字符的字符串。主机名称可以包括连字符但不能是第一个或者最后一个字符。默认值为BMC+MAC地址 (示例：MAC地址'00 1e 67 f9 2f b3'，预设主机名称会是'bmc-mac001e67f92fb3')。

接口

它列出了可用于管理服务器的接口。

MAC地址

设备的MAC地址（只读）。

IPMI 频道

显示 IPMI 通道号（只读）。

接口描述

显示网络接口控制器是专用于BMC/主机或是主机和BMC共用（只读）。

网络链接

显示局域网接口的网络链接状态（只读）。

使用DHCP来自动获取IP地址

选择启用 IPv4 DHCP 的选项。如果选择“自动获取 IP 地址（使用 DHCP)”选项，IP 地址、子网掩码、网关和 DNS 服务器输入字段将被禁用。

使用如下IP地址

选择配置静态 IPv4 地址的选项。如果选择“使用以下 IP 地址”选项，将启用 IP 地址、子网掩码、网关和 DNS 服务器输入字段来配置

IP地址

使用单选按钮选择 IP 分配的类型。如果配置静态 IP，请在给定字段中输入请求的 IP 地址、子网掩码、网关、主 DNS 和辅助 DNS 服务器。

  • - IP地址是由4个数字用点隔开如“xxx.xxx.xxx.xxx”。
  • - “xxx”范围从0到255
  • - 第一个“xxx”不能是0

子网掩码

显示设备的子网掩码。

默认网关

表示设备的网关详细信息。

优先的DNS服务器IP

显示设备的主 DNS 服务器。

备用的DNS服务器IP

设备的辅助 DNS 服务器。

操作
保存

点击“保存”按钮保存任何修改。

备注： 专用的管理网卡IP地址必须跟基板在不同的子网IP地址。


备注： 在尝试修改网络设置前，请关闭所有打开的远程控制台重定向会话。否则所有的会话将丢失。
