活动目录设置 - 帮助

要启用或配置 活动目录 服务器，请选中该复选框。

启用 活动目录 身份验证

选中复选框以启用/禁用 活动目录 身份验证。

通过SSL来对活动目录身份认证

选中复选框以启用/禁用基于 SSL 的活动目录身份验证。

启用 活动目录 需要 CA 证书和 LDAP 证书。

CA 证书有效期至 -- 表示 CA 证书的到期日期。

LDAP 证书有效期至 -- 它表示 LDAP 证书的到期日期。

IP 地址

活动目录服务器的IP地址。

  • - IP 地址由 4 个数字组成，用点分隔,如“xxx.xxx.xxx.xxx”。
  • - 'xxx' 范围从 0 到 255。
  • - 第一个 'xxx' 不能为 0。
  • - 支持主机名

绑定DN

活动目录 服务器的专有名称，例如 “cn=Manager, dc=my-domain, dc=com”。

绑定密码

活动目录服务器的认证密码; 密码长度必须至少为 4 个字符。

搜索基址

活动目录 服务器的搜索基址，例如 “dc=my-domain, dc=com”。

超时时间

活动目录 服务器超时时间

用户 ID 属性（可选）

活动目录 服务器的用户 ID

端口（可选）

活动目录服务器的端口号

活动目录 组配置

这是创建组配置的过程，其中包含名称、域、权限等多个详细信息

角色组名

输入角色组名称

角色组域

输入角色组域名

角色组权限

选择角色组权限

动作
保存

单击“保存”按钮以保存 活动目录 配置。

删除

单击“删除”按钮以删除 活动目录 组配置

当从表中选择配置的组时，删除按钮将可用
