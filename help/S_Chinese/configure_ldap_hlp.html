关
LDAP — 帮助

选择或者不选“LDAP认证”选项来启用或禁用LDAP。

LDAP认证

选择这个选项来启用LDAP认证，然后输入需要的信息来存取LDAP服务器。

通过SSL来对LDAP认证

选中该复选框以启用/禁用通过 SSL 的 LDAP 身份验证。

IP地址

LDAP服务器的IP地址。

  • - IP地址是由4个数字用点隔开如“xxx.xxx.xxx.xxx”。
  • - “xxx”范围从0到255
  • - 第一个“xxx”不能是0
  • - 支持主机名

绑定的DN

LDAP服务器的专有名称，比如 "cn=Manager, dc=my-domain, dc=com".

绑定的密码

为LDAP服务器认证密码，密码长度必须大于4个字符。

搜索基址

LDAP的搜索基址，比如 "dc=my-domain, dc=com".

用户 ID 属性（可选）

LDAP 服务器的用户 ID

配置LDAP用户组 - 帮助

组名称

角色组名称是 64 个字母数字字符的字符串。

允许使用特殊符号连字符和下划线。

组权限

选择角色组权限。这是为此角色组分配的权限级别

操作
保存

保存当前设定。

删除

从表中选择 LDAP 组并删除该组。
