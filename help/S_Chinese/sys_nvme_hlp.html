NVMe信息 — 帮助

本页显示每个英特尔®公司的NVMe驱动安装到一个热插拔驱动器插槽的状态信息。 每个组框显示中包含一个单一的NVMe驱动统计。 每个组框通过热插拔背板（HSBP) 编号和驱动槽编号识别驱动器。 只有五个条目能够在运行时改变状态：

型号:: 表示当前型号序列号

PCIe 0 链接速度: 它显示 PCIe 0 的链接速度。

PCIe 1 链接速度: 它显示 PCIe 1 的链接速度。

NVMe 通电: 指示 NVMe 驱动器的当前开机状态。该值只能在 NVMe 驱动器通电时显示。

需要 NVMe 重置： 指示 NVMe 驱动器是否报告需要重置驱动器的情况。

设备类： 它表示设备类的详细信息。

设备编程 Intfc: 它显示设备编程 Intfc 的详细信息。

端口 1 PCIe 链接活动：它显示 Port 1 PCIe Link Active 的详细信息.

设备位置: 它显示了设备位置的详细信息.

序列号: 它显示序列号的详细信息.

PCIe 0 链接宽度： 它显示了 PCIe 0 Link Width 的详细信息。

PCIe 1 链接宽度：它表示 PCIe 1 Link Width 的详细信息。

NVMe 功能： 它显示 NVMe 功能的详细信息。

端口 0 PCIe 链接活动： 它指示 Port 0 PCIe Link Active 状态的详细信息。

设备子类： 它显示设备子类的详细信息。

驱动寿命消耗： 它显示 Drive Life Consumed 的详细信息。

制造商：它显示制造商的详细信息。

固件版本： 它表示固件版本的详细信息。
